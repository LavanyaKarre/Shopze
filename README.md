# Shopze
Shopze with MERN Stack

## Setup Instructions for the Team

1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/shopez-mern.git

2. Set up the BACKEND:
   cd server
   npm install
   - Create a file named `.env` in the server folder
   - Copy everything from `.env.example` into it
   - Replace the placeholder values with the real ones (ask the team lead)

3. Set up the FRONTEND:
   cd client
   npm install

4. Run the BACKEND (in one terminal):
   cd server
   npm run dev

5. Run the FRONTEND (in another terminal):
   cd client
   npm run dev
